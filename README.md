＜制作目標＞
Proccesingを利用したArduinoオーケストラを作る

具体的には「班員分　-(マイナス)　1台」のArduinoを利用し、親機からの指示で子機が動くようなシステムを作る。
子機はそれぞれ楽器を担当し、同期のずれをできるだけ無くした状態にする。曲は輪唱曲から「きらきら星」を選択、
またそれぞれ楽器担当のArduinoも輪唱させる、輪唱のコードを書くのは親機のArduino。

またエンターテインメント性が必要であり、私たちの班はリズムゲームを制作した。
